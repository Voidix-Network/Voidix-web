# 社区行为准则

## 我们的承诺

我们致力于为所有人提供一个友好、安全、包容的社区环境。

## 行为标准

**鼓励的行为：**
- 尊重不同观点和经验
- 接受建设性批评
- 关注社区利益
- 友善对待其他成员

**不可接受的行为：**
- 骚扰、侮辱或人身攻击
- 发布他人隐私信息
- 不当的性语言或图像
- 其他不专业的行为

## 适用范围

本准则适用于所有项目空间，包括 GitHub 仓库、Issues、PR、社区讨论等。

## 举报

遇到违规行为请联系：conduct@voidix.net

我们会在 24 小时内确认收到，并在 7 个工作日内处理。

## 执行

违规后果视情节轻重，从警告到永久封禁不等。
